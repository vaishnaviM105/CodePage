Digit Count in Python

Problem Statement:

Write a code in any programming language that counts the digits in a number.

Code

    num = 7834
    count = 0
    while num != 0:
        num //= 10
        count += 1

    print("Number of digits: " + str(count))

Sample Input

7834

Sample Output

Number of digits: 4

Variables used in program

num, count

Complexity

O(n)
